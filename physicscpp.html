PhysicsCPP: modular physics demonstration library (+software) written in C++

PhysicsCPP is an educational tool (software and C++ library) for creation of physics demonstrations, which was presented at the '13th Panhellenic/International Conference "ICT in Education"'. It provides a basic graphical environment with an integrated, easy to understand custom UI library. It utilizes the GPLv3 copyleft license, meaning that contributions and pull requests are welcome, as well as providing very rich documentation, explaining the essence of the project clearly, easing its use.
